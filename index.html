Reference

Every way to steep a request

Eleven patterns, one import. Each card is a complete, runnable snippet.

01 · Basics

Send your first message

Pass a model and an api key — the provider is detected from the model name.

from TeaLow import TeaLow

ai = TeaLow(model="gemini-2.5-flash", api="YOUR_API_KEY")
print(ai.send("Hello!"))
02 · Config

Use environment variables

Skip api= entirely by exporting the provider's standard key variable.

# export OPENAI_API_KEY="sk-..."
from TeaLow import TeaLow
ai = TeaLow(model="gpt-4o-mini")
print(ai.send("Hello!"))
03 · Memory

Multi-turn conversation

Set a system_prompt once, then read every turn back from ai.history.

ai = TeaLow(model="claude-sonnet-4-6", api="KEY",
            system_prompt="You are a helpful pirate.")
ai.send("What's your name?")
for m in ai.history:
    print(m["role"], "->", m["content"])
04 · Streaming

Stream tokens as they arrive

Iterate over ai.stream() and print each chunk.delta live.

for chunk in ai.stream("Write a haiku about the ocean."):
    print(chunk.delta, end="", flush=True)
05 · Async

Async client, same shape

AsyncTeaLow mirrors the sync API for asyncio apps and servers.

import asyncio
from TeaLow import AsyncTeaLow

async def main():
    ai = AsyncTeaLow(model="gemini-2.5-flash", api="KEY")
    print(await ai.send("Hello from asyncio!"))
    await ai.close()

asyncio.run(main())
06 · Vision

Send an image along

Load a file with TeaLow.load_image() and pass it via images=.

image = TeaLow.load_image(path="photo.jpg")
response = ai.send("What is in this image?", images=[image])
print(response)
07 · Structured

JSON mode

Set json_mode=True and parse response.text straight into Python.

response = ai.send(
    "Return JSON with keys 'name' and 'age'.",
    json_mode=True,
)
data = json.loads(response.text)
08 · Generation

Generate an image

Point the model at an image generator like dall-e-3 and call generate_image().

ai = TeaLow(model="dall-e-3", api="KEY")
urls = ai.generate_image("A watercolor teapot on a low table")
print(urls[0])
09 · Transport

Headers, proxy & timeouts

Tune timeout, max_retries, proxies, and custom headers per client.

ai = TeaLow(
    model="gpt-4o-mini", api="KEY",
    timeout=30.0, max_retries=5,
    proxies={"https": "http://proxy.internal:8080"},
    headers={"X-Org-Id": "acme-corp"},
)
10 · Observability

Turn on logging

Call configure_logging() once at startup to see request-level detail.

from TeaLow import configure_logging
import logging
configure_logging(level=logging.DEBUG)
11 · Resilience

Handle errors gracefully

Catch RateLimitError and InvalidAPIKeyError specifically, or TeaLowError for everything else.

try:
    response = ai.send("Hello!")
except RateLimitError as e:
    print("Retry after:", e.retry_after)
except InvalidAPIKeyError:
    print("Key was rejected.")
